🎨 PDFカラフル変換ツール

モノクロのパンフレットを色鮮やかで高級感のある仕上がりに自動変換

✅ 主な機能

  • PDFファイルの自動カラー変換
  • 4種類のカラースキーム選択
  • グラフ・表の水色変換
  • 個別・一括ダウンロード機能
  • 完全オフライン動作

🚀 今すぐ使用する

GitHubからダウンロード

📋 セットアップ手順

1. リポジトリをクローン

git clone https://github.com/katou-maker-main/sachiko_20250715.git cd sachiko_20250715

2. 依存関係をインストール

pip install -r requirements.txt

3. アプリを起動

streamlit run app.py

4. ブラウザでアクセス

http://localhost:8501 にアクセスして使用開始！

🏢 チーム共有

社内チームで共有する場合：

  1. Windowsの場合: start_server.bat をダブルクリック
  2. チームメンバーは表示されたIPアドレスでアクセス
  3. 詳細は SETUP_GUIDE.md を参照

💡 技術仕様

  • フレームワーク: Streamlit
  • PDF処理: PyMuPDF
  • 画像処理: OpenCV, Pillow
  • 対応OS: Windows, Mac, Linux
  • Python: 3.8以上
